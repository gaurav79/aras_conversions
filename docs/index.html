gtsi-srv-gtr

GTSI GTR SERVICE

GTR Service

  • This service exposes various endpoints for managing data in GTR tables of ARAS.

example api requests -

  • GET - http://localhost:3000/api/v1/gtr/{id?}
  • POST - http://localhost:3000/api/v1/gtr
  • PUT - http://localhost:3000/api/v1/gtr/{id}
  • DELETE - http://localhost:3000/api/v1/gtr/{id}

Installation

  • npm i

Important Note

  • If working with NodeJS 10.0.0, Kindly delete the package.lock.json file then try npm install

Run

  • gulp build (build ts files)
  • gulp test (run mocha tests)
  • gulp tslint (run tslint)
  • gulp watch (watch ts files)
  • gulp (start the application using gulp)

Features

  • Project Structure - Feature oriented
  • Hapijs - REST Api
  • Swagger - documentation
  • Unit Tests - chai + sinon + mocha
